# JavaScript版本

随着Mozilla基金会继续开发 JavaScript，添加新的特性、关键字和语法，JavaScript 的版本号继续递增……

只有 Netscape/Mozilla浏览器才遵循这种编号模式，如JavaScript版本 1.0……1.8……这个样子
